MP4 → MP3 최대 음질 변환기

MP4 영상을 브라우저에서 바로 MP3로 변환합니다. 서버 업로드 없이 처리되며, 가능한 한 높은 음질(기본 320kbps CBR)로 추출하도록 구성했습니다.

MP4 파일을 올려주세요 클릭하거나 여기로 드래그하세요.
지원 형식: MP4, MOV, M4V, WEBM 등 영상 파일

최대 음질은 원본 음원 품질을 넘어설 수는 없습니다. 원본 영상의 오디오가 이미 저음질이면, MP3를 320kbps로 저장해도 소스 이상의 품질이 되지는 않습니다. 그래도 재인코딩 품질 손실을 최소화하는 방향으로 셋업했습니다.
이 툴은 브라우저 보안 정책상 file:// 로 직접 열면 동작하지 않을 수 있습니다.
반드시 로컬 서버(http://localhost) 또는 웹 서버 환경에서 실행하세요.

빠른 실행 예시:
- Python: python -m http.server 8000
- VS Code: Live Server 확장 사용
- 접속 주소: http://localhost:8000
